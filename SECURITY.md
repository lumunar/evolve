# Security Policy

We take the security of our libraries seriously. If you find a security vulnerability, please report it to us so we can resolve it quickly.

### Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. Reporting vulnerabilities publicly can expose projects to risk before a fix is available.

Instead, please report vulnerabilities privately by emailing us:

*   Send your report to **community@lumunar.com**
*   Include a detailed description of the vulnerability
*   Provide steps or code to reproduce the issue

We will review your report and respond within forty eight hours to coordinate a resolution.
